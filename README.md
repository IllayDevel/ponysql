# PonySQLDB

Little RDBMS engine on Java.

## Build

PonySQLDB now uses a modern Maven build and targets Java 21 bytecode.

Requirements:

- JDK 21 or newer for local development
- Maven 3.9 or newer

Common commands:

```sh
mvn verify
mvn package
```

The runnable JAR entry point is `com.pony.runtime.PonyDBMain`.
